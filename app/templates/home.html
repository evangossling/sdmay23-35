The objective of our web application is to provide an end-to-end system that will provide such functionalities for the users interested in visualizing locations related to popularity cascades in the context of scientific paper citations. We use the machine learning model: HINTS Citation Time Series Prediction. HINTS uses their machine learning model to obtain three values: η, μ, σ.

Using these obtained values in the above equation, we are able to predict the expected number of citations of x years, as shown in the graph below.

Part of our purpose, is to integrate this machine learning model into it. We combine this with the APS dataset, of which the model is trained on, to map the locations associated with the paper and the locations of papers that cited the original paper into our Mapbox map.
